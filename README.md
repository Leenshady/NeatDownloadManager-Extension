# NeatDownloadManager-Extension
官方的 NeatDownloadManager Extension 没办法屏蔽某一个网站的下载导致屏幕上出现一堆下载框。
这个扩展在原有的扩展中添加了屏蔽网站的功能。

# 使用方法
在目标网站右键选择 Block downloads from this website 即可。
如果要打开下载在目标网站右键选择 Download by NeatDownloadManager。
